Politique de confidentialité — Météo Québec

Dernière mise à jour : 18 juin 2026

Cette application ne collecte aucune donnée personnelle. Aucun compte n'est requis.

Données stockées localement

L'application enregistre vos préférences (ville sélectionnée, code postal) uniquement sur votre appareil, via le stockage local du navigateur. Ces données ne quittent jamais votre appareil.

Données météo

Les prévisions sont fournies par Open-Meteo, un service open source. Lors d'une requête météo, votre appareil contacte nos serveurs, qui transmettent la demande à Open-Meteo. Aucune donnée personnelle n'est incluse dans ces échanges.
